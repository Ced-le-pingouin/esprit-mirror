===============
JAVA dans Esprit
================

Deux modules d'Esprit sont actuellement en Java : awareness et chat.
Chacun fonctionne sur un modèle client/serveur.

La partie client est déjà intégrée à Esprit dans l'arborescence principale.

La partie serveur doit être lancée sur le serveur pour que ces modules
soient fonctionnels. Il est pour cela possible d'utiliser les scripts du
répertoire scripts/. Dans ce cas, il faut éditer les chemins d'accès dans
l'entête de ces fichiers.
