
INSTALLATION ET MISE A JOUR DES DONNEES SQL

Dans chaque répertoire, les fichiers *.sql sont lus selon l'ordre alphabétique.

Les fichiers *.sql sont lus :
 - à l'installation, dans ce répertoire uniquement ;
 - à la mise à jour, dans le sous-répertoire mises_a_jour.

Les anciennes mises à jour sont disponibles dans les répertoires
mises_a_jour/{version_de_départ}_vers_{version_de_destination}/ .
