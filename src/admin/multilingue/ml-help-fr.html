Introduction

Cette aide décrit l'installation du support multilingue pour plusieurs systèmes d'exploitation. Le support inclut deux points :

  • affichage : mise à disposition des polices de caractères nécessaires ;
  • saisie : mise en place des méthodes de saisie nécessaires.

Windows XP

L'installation du support multilingue se fait en deux étapes : une première effectuée par l'administrateur (si elle n'a pas été faite à l'installation de Windows), et une deuxième étape de configuration effectuée par chaque utilisateur pour son compte.

Installation du support multilingue

Cette étape nécessite obligatoirement les droits d'administrateur. Il s'agit d'ajouter sur le système les polices de caractère et les utilitaires de saisie nécessaires.
Ouvrez le Panneau de configuration, et choisissez Paramètres régionaux et linguistiques, puis l'onglet Langues.
Dans la fenêtre qui s'ouvre, cochez les deux cases, et validez. Attention, Windows attendra le CD d'installation, et a besoin de 200 Mo d'espace libre sur le disque dur !

Configuration utilisateur

Cette étape doit être effectuée par chaque utilisateur pour configurer son propre compte système. Il s'agit d'ajouter les méthodes de saisie qu'il utilisera.
Ouvrez le Panneau de configuration, et choisissez Paramètres régionaux et linguistiques, puis l'onglet Service d'entrée de texte, puis cliquez sur le bouton Détails.
Grâce au bouton Ajouter..., vous pouvez maintenant sélectionner dans une longue liste les langues qui vous intéressent. À chaque langue correspondent plusieurs méthodes d'entrée, mais en général le choix par défaut convient. Chaque langue ajoutée sera accessible à partir de la barre d'état, généralement placée en bas à droite du bureau Windows.

Windows 98/Me

Malheureusement ces systèmes sont trop vieux pour pouvoir permettre un fonctionnement parfait des langues étrangères sous Windows. On peut toutefois obtenir un affichage normal et, dans une moindre mesure, une saisie des langues non latines.

Saisie

Lancer Windows Update et choisissez Prise en charge des langues internationales. Parmi les langues qui vous sont alors proposées, choisissez celle qui vous est nécessaire. Il est recommandé de mettre à jour Internet Explorer. A défaut de connection Internet, on peut se servir du CD de Windows : Panneau de configuration -> Ajouter/Enlever des programmes -> Programmes Windows. Cliquez alors sur Support multilingue puis sur Détails.

Ensuite, dans le panneau de configuration, cliquer sur l'icone Clavier puis sur l'onglet Langue. Il ne reste plus qu'à ajouter la langue désirée grâce au bouton Ajouter. Cocher Activer l'indicateur sur la barre de tâche vous permettra par la suite de changer de mode de saisie au clavier en utilisant une icone de la barre des tâches, en bas de l'écran.

Affichage

A priori, la configuration de la saisie entraine l'installation des polices de caractères nécessaires et le navigateur web les utilisera directement. Si toutefois certaines pages ne s'affichent pas correctement dans Internet Explorer, vous pouvez chercher à configurer les polices utilisées avec Options Internet -> Fontes.

Mac OS X

Mac OS X est déjà configuré pour pouvoir afficher des langues non-latines, il faut toutefois activer la méthode de saisie adaptée à la langue. Pour cela, dans le menu Pomme, sélectionner Préférences système... puis cliquer sur l'icone International. L'onglet Saisie (parfois intitulé Menu Clavier) vous permet alors de choisir plusieurs configurations du clavier. Si vous cochez la case Afficher le menu Saisie dans la barre de menus, le menu Saisie apparaît dans le coin supérieur droit de la barre de menus.

Linux

Affichage

La plupart des distributions récentes installent automatiquement les fontes nécessaires. A défaut, installer les polices nécessaires, soit spécialement adaptées à la langue voulue, soit générales (comme la plupart des dontes Unicode).

Saisie

La démarche à suivre dépend de votre distribution. Par exemple pour Ubuntu, ces réglages sont accessibles dans système/préférences/clavier. Si vous utilisez KDE, le Centre de configuration permet de sélectionner plusieurs méthodes de saisie au clavier. Gnome propose un équivalent dans son interface de configuration. A noter que pour les langues asiatiques, vous serez peut-être amené à installer manuellement une méthode de saisie comme SCIM avant de pouvoir configurer votre gestionnaire de bureau.
