Bienvenue sur Esprit !



Pour découvrir Esprit, vous vous invitons à parcourir les différentes activités de ce séminaires et leurs outils associés.

Vous pouvez également consultez le module "Comment suivre une formation sur Esprit?" qui se trouve en page d'accueil. Vous devez pour cela revenir à la page d'accueil en cliquant sur "Quitter".

Vous pouvez aussi consulter le manuel de l'utilisateur qui est à votre disposition dans le menu de gauche.

ATTENTION : Pour pouvoir utiliser le chat et l'awarness, vous devez demander l'ouverture des ports suivants

  • N° port du serveur chat : 2555
  • N° port du serveur awareness :2501
Vous trouverez le plug-in nécessaire pour Explorer sur le site de Macromedia™ en suivant le lien http://macromedia.com/shockwave/download/triggerpages_mmcom/flash-fr.html

Ces pages été développées pour des navigateurs de nouvelle génération (Netscape 6.2 ou Explorer 5.01) ou d'une génération ultérieure. Si vous ne possédez pas un tel navigateur, vous devez l'installer en vous rendant sur le site du producteur.

Si vous deviez rencontrer quelques difficultés techniques, vous pouvez nous en faire part en déposant un message dans le forum du groupe.

Nous vous souhaitons une agréable découverte !
