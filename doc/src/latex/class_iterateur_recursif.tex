\subsection{Description détaillée}
constante interne à des fins de vérification 

Itérateur \char`\"{}décorateur\char`\"{} qui permet de parcourir automatiquement non seulement les éléments d'un itérateur, mais aussi les enfants de ce dernier lorsqu'il y en a. L'itérateur \char`\"{}décoré\char`\"{} doit implémenter l'interface \doxyref{IterateurComposite}{p.}{class_iterateur_composite} pour que ça fonctionne

\subsubsection{\setlength{\rightskip}{0pt plus 5cm}IterateurRecursif::suiv ()}\label{class_iterateur_recursif_05e95b2a82b1bcbb81a954422a3da897}


Déplace l'itérateur d'une position vers l'avant. 

Dans le cas de notre itérateur récursif, cela veut également dire que si un élément d'itérateur est lui-même \char`\"{}traversable\char`\"{}, l'itération continue également dans les enfants de cet élément (=sous-itérateur). De même, un appel à \doxyref{suiv()}{p.}{class_iterateur_recursif_05e95b2a82b1bcbb81a954422a3da897} lorsqu'un sous-itérateur est en position finale retourne le prochain élément de l'itérateur parent qui avait été \char`\"{}laissé\char`\"{} (si ce dernier en contient encore, évidemment)
