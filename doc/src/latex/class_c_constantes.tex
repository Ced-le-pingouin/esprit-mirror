Crée un fichier de constantes représentant les termes traduisibles. 

Cette fonction doit être appelée lorsque de nouveaux termes à traduire ont été ajoutés dans la DB, sinon leur id sera inaccessible en PHP

\begin{Desc}
\item[Paramètres:]
\begin{description}
\item[{\em v\_\-sNomFichier}]le nom du fichier de constantes à créer. Si {\tt v\_\-sNomFichier} n'est pas spécifié, le fichier aura le même nom que la table utilisée pour la langue (voir \doxyref{CConstantes()}{p.}{class_c_constantes_8ee22843123851c029ef232a8d97ea38}) \end{description}
\end{Desc}
