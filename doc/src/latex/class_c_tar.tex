\subsection{Description détaillée}
Gestion d'archives GNU TAR, utilisant les fonctions de la bibliothèque PclTar. 

\begin{Desc}
\item[{\bf À faire}]Cette classe est incomplète, elle ne gère pas la décompression des archives, uniquement la compression. De plus, seules des archives gzippées sont créées \end{Desc}

\subsubsection{\setlength{\rightskip}{0pt plus 5cm}CTar::ajouterEntreesDsListe (\$ {\em v\_\-asEntrees}, \/  \$ {\em v\_\-bEffacerAnciennes} = {\tt FALSE}, \/  \$ {\em v\_\-bCreer} = {\tt FALSE})}\label{class_c_tar_d43036987c5d5649572d42bfc6839e48}


Ajoute une entrée (fichier ou dossier) à la liste des entrées qui constitueront le fichier compressé. 

\begin{Desc}
\item[Paramètres:]
\begin{description}
\item[{\em v\_\-asEntrees}]la liste des entrées à inclure à l'archive, sous forme de tableau de chaînes de caractères, chacune représentant le chemin d'un fichier/dossier à inclure \item[{\em v\_\-bCreer}]si {\tt true}, l'archive est créée immédiatement, et le code de retour de cette opération est retourné\end{description}
\end{Desc}
\begin{Desc}
\item[Renvoie:]si {\tt v\_\-bCreer} est à {\tt true}, le code de retour de l'opération de création de l'archive est retourné. Sinon, la valeur retournée est {\tt 1} \end{Desc}
