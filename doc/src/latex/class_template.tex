Retourne une \char`\"{}variable de template\char`\"{}, càd un bloc entre balises [bloc+][bloc-]. 

Par exemple si le template contient un segment \char`\"{}[bloc+]contenu[bloc-]\char`\"{}, un appel {\tt defVariable(\char`\"{}bloc\char`\"{})} renvoie le texte {\tt \char`\"{}contenu\char`\"{}} 

\begin{Desc}
\item[Paramètres:]
\begin{description}
\item[{\em looptag}]le nom du bloc dont les balises entourent la variable de template\end{description}
\end{Desc}
\begin{Desc}
\item[Renvoie:]la variable trouvée. S'il n'existe pas de variable définie par des blocs [looptag+][lopptag-], renvoie {\tt null} \end{Desc}

Remplace \char`\"{}simplement\char`\"{} des morceaux de template par d'autres. 

Utilisé pour inclure des variables dans le template qui seront remplacées en PHP, par exemple un appel {\tt remplacer(\char`\"{}\{utilisateur\}\char`\"{}, \$oProjet-$>$oUtilisateur-$>$retPseudo())} remplace toutes les chaînes {\tt \char`\"{}\{utilisateur\}\char`\"{}} trouvées dans le template par le pseudo de l'utilisateur connecté

\begin{Desc}
\item[Paramètres:]
\begin{description}
\item[{\em in}]la chaîne de caractère à trouver dans le template \item[{\em out}]la chaîne par laquelle on veut remplacer toutes les occurences de {\tt in} \end{description}
\end{Desc}
