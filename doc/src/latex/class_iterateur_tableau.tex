\subsection{Description détaillée}
Sous-classe d'IterateurAbstrait, qui permet d'effectuer des itérations sur un tableau, mais implémente en plus l'interface \doxyref{IterateurBidir}{p.}{class_iterateur_bidir} pour fournir un itérateur bidirectionnel. 

\begin{Desc}
\item[Note:]Pour le moment, il n'y a pas d'implémentation spécifique pour \doxyref{rechercher()}{p.}{class_iterateur_abstrait_64c0cceb7d56d7cb815b8bdb8192ed99}, elle est récupérée d'IterateurAbstrait, et est donc \char`\"{}générique\char`\"{} \end{Desc}
