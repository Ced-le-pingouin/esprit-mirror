\subsection{Description détaillée}
Classe abstraite qui sert de base pour des décorateurs d'Iterateur. 

Elle n'a pas d'utilité tant qu'elle n'est pas sous-classée, mais elle permet de ne pas recopier dans chaque décorateur des \char`\"{}\$this $\rightarrow$ oItr $\rightarrow$ ...()\char`\"{} pour transférer chaque appel du décorateur vers son itérateur interne (par exemple \doxyref{IterateurFiltre}{p.}{class_iterateur_filtre} dérive de cette classe) \subsection*{Fonctions membres publiques}
