Enlève une personne de toutes les équipes rattachées à un élément d'un niveau spécifié (formation, module, etc), MAIS aussi des équipes rattachées aux éléments enfants de celui spécifié.

Paramètres:
v_iIdPers l'id de la personne à effacer des équipes
v_iNiveau le numéro représentant le type d'élément auquel doivent être rattachées les équipes dont on veut enlever la personne (voir les constantes TYPE_)
v_iIdNiveau l'id de l'élément pour lequel on veut trouver les équipes attachées afin d'en effacer la personne passée en v_iIdPers. La signification de cet id dépend du paramètre v_iTypeNiveau
Renvoie:
true si l'id de la personne passé en paramètre est valide, false dans le cas contraire

CEquipe_Membre::init (  ) 

Initialise un tableau de personnes (aoMembres) sous forme d'objets CPersonne, qui sont les membres de l'équipe actuellement représentée par iId (qui peut être initialisé par le constructeur CEquipe_Membre()).

Initialise un tableau de personne (aoMembres) avec les membres des équipes répondant à certains critères (équipe rattachée à une formation, un cours, etc.

particuliers), ou avec les personnes *non*-membres d'équipes spécifiques

Paramètres:
v_iNiveau le numéro représentant le type d'élément pour lequel on veut récupérer les (non-)membres d'équipes, càd formation, module, rubrique, activité, sous-activité (voir les constantes TYPE_)
v_iIdNiveau l'id de l'élément pour lequel on veut récupérer les (non-)membres d'équipes. Sa signification dépend du paramètre v_iTypeNiveau
v_bAppartenirEquipe si true, ce sont les membres des équipes répondant aux critères passés en paramètres v_iNiveau et v_iIdNiveau qui sont retournés. Si false, ce sont les personnes n'appartenant *pas* à ces équipes
Renvoie:
le nombre de membres trouvés

CEquipe_Membre::verifMembre ( $  v_iIdPers  ) 

Vérifie qu'un personne est membre d'équipes spécifiques.

Ces équipes (et donc les membres) ont été déterminées lors d'appels au constructeur CEquipe_Membre(), à init(), ou à initMembresDe(), qui auront donc initialisés au préalable le tableau aoMembres. Il est à noter que dans le cas de la dernière fonction, il est possible de rechercher les *non*-membres d'équipes spécifiées

Paramètres:
v_iIdPers l'id de la personne dont on veut vérifier l'appartenance (ou la non-appartenance) à des équipes spécifiques
Renvoie:
true si la personne spécifiée est "membre" (existe dans le tableau aoMembres); false dans le cas contraire ou dans le cas ou l'id de la personne n'est pas valable ou le tableau aoMembres n'est pas correctement initialisé
