Description détaillée

Classe utilitaire pour créer et obtenir des informations sur un "élément de formation", càd des objets de CFormation, CModule, etc.

À faire:
Cette classe devrait normalement devenir une classe de base ("mère") dont devraient hériter les classes qui représentent un élément de formation
