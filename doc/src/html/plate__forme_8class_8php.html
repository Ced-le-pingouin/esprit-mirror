Documentation des fonctions

retListeStatuts ( $  v_sGenre = "F"  ) 

Retourne un tableau de chaînes de caractères représentant la version textuelle d'un statut ouvert/fermé/etc pour les éléments des formations.

Paramètres:
v_sGenre la chaîne F pour obtenir la version au féminin du mot, M pour la version au masculin
Renvoie:
le tableau contenant les versions texte des statuts, avec pour indices les constantes STATUT_ correspondantes
