Exécute une requête SQL, et en cas de réussite enregistre le handle du résultat dans le tableau ahResult.

Paramètres:
v_sRequete le texte de la requête, sans point-virgule final. Une seule requête à la fois donc
v_bAfficher si true, le texte de la requête est affiché dans la page (debug)
Renvoie:
l'indice du handle de résultat dans le tableau ahResult en cas de réussite de la requête, false en cas d'échec (erreur de syntaxe ou autre)

CBddMySql::retEnregPrecis ( $  v_iNumResult = NULL,
$  v_iNumEnreg = 0 
)

Retourne le premier champ d'un enregistrement donné dans un résultat de requête.

Paramètres:
v_iNumResult l'indice du résultat de requête à utiliser (voir retNbEnregsDsResult())
v_iNumEnreg l'indice de l'enregistrement dans le résultat (le premier = 0)
Renvoie:
le premier champ de l'enregistrement spécifié, dans le résultat de requête voulu
Note:
Cette fonction est limitée au premier champ, elle est surtout utilisée pour les requêtes SELECT dont le résultat n'a qu'un enregistrement et un seul champ non nommé, comme SELECT MD5("blah") ou SELECT COUNT(*) FROM Table

Paramètres:

Renvoie:

CBddMySql::retNbEnregsDsResult ( $  v_iNumResult = NULL  ) 

Retourne le nombre d'enregistrements récupérés par une requête.
