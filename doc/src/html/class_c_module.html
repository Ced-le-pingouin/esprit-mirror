CModule::initMembres ( $  v_bAppartenirEquipe = TRUE,
$  v_bAutoInscrit = TRUE,
$  v_iSensTri = TRI_CROISSANT 
)

Initialise un tableau avec les étudiants inscrits à la formation.

Paramètres:
v_bAppartenirEquipe si true (par défaut) le tableau est rempli par les personnes qui appartiennent à une équipe de ce module, si false voir paramètre v_bAutoInscrit
v_bAutoInscrit si true (par défaut) le tableau est rempli par les personnes qui sont inscrites à la formation et qui n'appartiennent pas à une équipe de ce module. Utilisé lorsque les personnes sont automatiquement inscrites aux modules de la formation. Si false il est rempli par les personnes qui sont inscrites au module et qui n' appartiennent pas à une équipe de ce module
v_iSensTri indique si un tri doit être effectué ainsi que son sens (croissant par défaut)
Renvoie:
le nombre de personnes(étudiants) insérées dans le tableau
