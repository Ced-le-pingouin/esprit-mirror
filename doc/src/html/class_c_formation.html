CFormation::initModules ( $  v_iIdPers = 0,
$  v_iIdStatutUtilisateur = NULL,
$  v_bRechStricte = FALSE 
)

Initialise un tableau contenant la liste des modules de la formation.

Paramètres:
v_iIdPers le numéro d'identifiant de la personne
v_iIdStatutUtilisateur l'id du statut de la personne utilisé comme critère pour la recherche. La signification de ce paramètre est modifiée par v_bRechStricte. Si le paramètre n'est pas fourni, tous les modules de la formation seront récupérés
v_bRechStricte si true et v_iIdStatutUtilisateur fourni, alors seul les modules pour lesquels la personne a exactement le statut demandé seront récupérés. Si false, les modules pour lesquels la personne a un statut inférieur ou égal à celui demandé seront également récupérés
Renvoie:
le nombre de module insérés dans le tableau

Référencé par copierModules(), effacerModules(), et retElementsEnfants().
