Référence de la classe CConstantes

Description détaillée

Classe permettant de récupérer des constantes 'texte' (consignes, messages, etc) dans la base de données.

CConstantes::creerFichierConstantes ( $  v_sNomFichier = NULL  ) 

Crée un fichier de constantes représentant les termes traduisibles.

Cette fonction doit être appelée lorsque de nouveaux termes à traduire ont été ajoutés dans la DB, sinon leur id sera inaccessible en PHP

Paramètres:
v_sNomFichier le nom du fichier de constantes à créer. Si v_sNomFichier n'est pas spécifié, le fichier aura le même nom que la table utilisée pour la langue (voir CConstantes())

Récupère un texte traduit sur base de son id.

Paramètres:
v_iId l'id du texte à chercher
v_bConversionHtml si true, remplace les caractères qui le nécessitent par des entités HTML dans le texte retourné
Renvoie:
