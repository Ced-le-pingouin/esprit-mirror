Description détaillée

Gestion d'archives PKZIP, utilisant les fonctions de la bibliothèque PclZip.

À faire:
Cette classe est incomplète, elle ne gère pas la décompression des archives, uniquement la compression. De plus, il faudrait "synchroniser" cette classe avec CTar (objet de base pour la (dé)compression puis héritage de chacune des deux classes ?), notamment au niveau des valeurs de retour

CZip::ajouterEntreesDsListe ( $  v_asEntrees,
$  v_bEffacerAnciennes = FALSE,
$  v_bCreer = FALSE 
)

Ajoute une entrée (fichier ou dossier) à la liste des entrées qui constitueront le fichier compressé.

Paramètres:
v_asEntrees la liste des entrées à inclure à l'archive, sous forme de tableau de chaînes de caractères, chacune représentant le chemin d'un fichier/dossier à inclure
v_bCreer si true, l'archive est créée immédiatement, et le code de retour de cette opération est retourné
Renvoie:
si v_bCreer est à true, le code de retour de l'opération de création de l'archive est retourné. Sinon, la valeur retournée est 1

Constructeur.

Paramètres:
v_sCheminArchive le chemin complet de l'archive qu'on désire créer
v_asEntrees un tableau contenant les chemins de fichiers ou dossiers à compresser. Si null (par défaut), il faudra ajouter des entrées plus tard à l'aide de la fonction ajouterEntreesDsListe(), et le paramètre v_bCreer ne sera donc pas pris en compte (il vaudra false)
v_bCreer si true, crée immédiatement l'archive. Le code de retour de cette opération est stocké dans la propriété iCodeRetour
