Constantes - types d'erreurs
Celles-ci reprennent les constantes E_USER_ de PHP.

Plus tard, il faudrait qu'elles deviennent des constantes de classe en laissant tomber le préfixe ERREUR_ au passage, mais les constantes de classe n'existent qu'en PHP 5

enum  ERREUR_AVERT
 l'erreur correspond aux warnings de PHP => pas d'arrêt du script mais elles sont affichées par défaut
enum  ERREUR_FATALE
 l'erreur correspond aux erreurs fatales de PHP, elle provoquera donc l'arrêt du script
enum  ERREUR_NOTE
 l'erreur correspond aux notices de PHP => je pense que par défaut elles ne sont pas affichées
