Description détaillée

Attributs publics

 $aFichiersSel
 Tableau des fichiers sélectionnés (formulaire).
 $bDezipperFichierDepose
 Lors de l'action Déposer un fichier, indique si on doit dézipper s'il s'agit d'un fichier zippé.
 $oDossierCourant
 Objet FichierInfo représentant le dossier actif (panneau Contenu).
 $oDossierRacine
 Objet FichierInfo représentant le dossier racine (panneau Arborescence).
 $oFichierARenommer
 Lors de l'action Renommer, objet représentant le fichier/dossier à renommer.
 $oFichierATelecharger
 Lors de l'action télécharger, objet représentant le fichier à télécharger.
 $oPressePapiers
 Objet représentant le presse-papiers.
 $sDossierACreer
 Lors de l'action Créer un dossier, objet représentant ce dossier.
 $sFichierDeposeNom
 Lors de l'action Déposer un fichier, nom de ce dernier.
 $sFichierRenomme
 Nouveau nom pour le fichier/dossier à renommer.
 $sFiltreFichiers
 Regex représentant les noms de fichiers/dossiers sur lesquels les actions sont interdites (non défini ici, destiné aux classes filles).
