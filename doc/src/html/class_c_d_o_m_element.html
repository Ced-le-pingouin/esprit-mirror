Constructeur.

Crée un élément DOM

Paramètres:
name le nom de l'élément à créer
value la valeur à attribuer à l'élément (A TESTER)
namespaceURI l'URI d'un espace de nom pour créer l'élément dans un espace de nom spécifique (INCOMPLET)
À faire:
L'attribution immédiate d'une valeur utilise la méthode setContent() des objets DOM PHP4, cela semble fonctionner pour créer des éléments XML (balises) avec un contenu texte, mais je ne sais pas si d'autres utilisations sont possibles
À faire:
Pour l'instant la création dans un espace de nom (3è paramètre) n'est pas implémentée
