Liste de tous les membres

Description détaillée

Classe abstraite qui sert de filtre (sous forme de décorateur) à des objets dérivés de Iterateur.

La méthode abstraite accepter() doit être redéfinie dans les sous-classes, de façon à déterminer si un élément est considéré comme "acceptable" ou pas selon le filtre utilisé.

L'algorithme utilisant accepter() pour retourner ou pas un élément de l'itérateur par la méthode courant() est quant à lui implémenté ici, et ne doit normalement pas être modifié
