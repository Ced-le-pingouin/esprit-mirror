Liste de tous les membres

Description détaillée

Classe de base pour les "traverseurs".

Ceux-ci permettent de traverser une formation et tous ses descendants, càd modules, rubriques, etc, et d'effectuer des actions à chaque étape, en disposant des infos sur l'étape courante (id, nom, etc de la formation, module, etc; fichiers associés)
