Description détaillée

Documentation des fonctions membres

FichierAcces::defChemin ( $  v_sChemin  ) 

voir FichierInfo::defChemin().

Cette surcharge de la méthode d'origine ferme le fichier représenté par l'objet s'il était ouvert, avant de modifier le chemin de l'objet

Réimplémentée à partir de FichierInfo.

Références estOuvert(), et fermer().

FichierAcces::ecrire ( $  v_sContenu  ) 

Ecrit des données dans le fichier représenté par l'objet.

Ce fichier doit être ouvert

Paramètres:
les données à écrire dans le fichier, sous forme de chaîne;
Renvoie:
le nombre d'octets effectivement écrits dans le fichier, ou false si une erreur se produit

Référencé par ecrireTout().

FichierAcces::ecrireTout ( $  v_sContenu  ) 

Ecrit des données en une seule fois dans le fichier représenté par l'objet courant.

Le fichier ne doit pas être ouvert préalablement

Références ecrire(), FichierInfo::existe(), fermer(), ouvrir(), et Erreur::provoquer().

FichierAcces::estOuvert (  ) 

Indique si le fichier actuellement représenté par l'objet est ouvert.

Renvoie:
true si le fichier est ouvert, false sinon

Référencé par defChemin().

FichierAcces::fermer (  ) 

Ferme le fichier représenté par l'objet courant.

Renvoie:
true si la fermeture s'est bien déroulée, false en cas d'erreur

Référencé par defChemin(), et ecrireTout().

FichierAcces::ouvrir ( $  v_iMode  ) 

Ouvre le fichier représenté par l'objet courant pour pouvoir y effectuer des opérations de lecture ou écriture.

Paramètres:
v_iMode le mode d'ouverture du fichier, qui doit être l'une des constantes FICHIER_MODE_...
Renvoie:
le handle du fichier ouvert, ou false si l'ouverture a échoué

Références FichierInfo::retChemin().

Référencé par ecrireTout().
