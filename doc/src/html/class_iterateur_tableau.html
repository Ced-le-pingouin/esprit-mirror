Description détaillée

Sous-classe d'IterateurAbstrait, qui permet d'effectuer des itérations sur un tableau, mais implémente en plus l'interface IterateurBidir pour fournir un itérateur bidirectionnel.

Note:
Pour le moment, il n'y a pas d'implémentation spécifique pour rechercher(), elle est récupérée d'IterateurAbstrait, et est donc "générique"
