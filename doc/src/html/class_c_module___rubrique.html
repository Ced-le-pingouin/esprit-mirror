CModule_Rubrique::initMembres ( $  v_bAppartenirEquipe = TRUE,
$  v_bAutoInscrit = TRUE,
$  v_iSensTri = TRI_CROISSANT 
)

Initialise un tableau avec les étudiants inscrits à la formation.

Paramètres:
v_bAppartenirEquipe si true (par défaut) le tableau est rempli par les personnes qui appartiennent à une équipe de cette rubrique, si false voir paramètre v_bAutoInscrit
v_bAutoInscrit si true (par défaut) le tableau est rempli par les personnes qui sont inscrites à la formation et qui n'appartiennent pas à une équipe de cette rubrique. Utilisé lorsque les personnes sont automatiquement inscrites aux rubriques de la formation. Si false il est rempli par les personnes qui sont inscrites à cette rubrique et qui n'appartiennent pas à une équipe de cette rubrique
v_iSensTri indique si un tri doit être effectué ainsi que son sens (croissant par défaut)
Renvoie:
le nombre de personnes(étudiants) insérées dans le tableau

CModule_Rubrique::retListeStatuts (  ) 

Retourne un tableau contenant la liste des statuts possibles d'une rubrique.

Cette liste se présente elle-même sous forme de tableaux, contenant chacun un statut possible de la rubrique (constante STATUT_), l'intitulé de ce statut, et une valeur booléenne indiquant si la rubrique courante possède ce statut actuellement

Renvoie:
le tableau contenant la liste des statuts possibles d'une rubrique
